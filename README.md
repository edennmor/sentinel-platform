Backend service built with Node.js and Express, serving as the foundation for a scalable security and monitoring platform.

## Description
This project implements a basic backend server that exposes a REST API endpoint, handles JSON requests, and enables cross-origin communication.  
It is designed as a clean starting point for further development such as authentication, database integration, and advanced API logic.

## Features
- Express-based HTTP server
- Health check API endpoint
- CORS and JSON middleware
- Ready for extension and scaling

## Tech Stack
- Node.js
- Express.js
- JavaScript
- Git & GitHub

## API
GET /api/health  
Returns server status and current timestamp.

## Running the Project

Install dependencies:
npm install

Start the server:
node server.js

Server runs on:
http://localhost:4000
